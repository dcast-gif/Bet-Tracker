# Bet Tracker

Bet Tracker is an application that allows users to upload screenshots of sports betting slips and receive live progress updates as events unfold.

---

## Features (Planned)

- 📸 Upload betting slip screenshots
- 🤖 AI extracts all bets from the screenshot
- ⚽ Automatically matches bets to live fixtures
- 📊 Tracks bet progress in real time
- 🔔 Sends push notifications when bets progress
- 💾 Stores betting history
- 📈 Shows live dashboard of all active bets

---

## Example

User uploads this bet:

Uruguay vs Spain

Market:
Over 2.5 Goals

Current score:

Uruguay 0-1 Spain

Bet Tracker displays:

Progress: 1 / 3 Goals

Needs 2 more goals.

If Spain score again:

Progress: 2 / 3 Goals

Needs 1 more goal.

If Uruguay score:

✅ Bet Won
